# AgendaKu

## Overview

AgendaKu adalah aplikasi web untuk membantu pengguna mengelola aktivitas harian secara sederhana, cepat, dan nyaman digunakan.

Fokus utama aplikasi adalah produktivitas tanpa fitur yang berlebihan.

Targetnya adalah pelajar, mahasiswa, pekerja, dan siapa saja yang ingin mengatur aktivitas harian.

---

## Goal

Membuat aplikasi To Do List yang:

- Cepat
- Modern
- Responsive
- Mudah digunakan
- Mudah dikembangkan
